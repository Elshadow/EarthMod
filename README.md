# EarthMod
可以将OSM地图的瓦片信息贴在世界中草地层上方(第五层)
可选学校信息依赖于运营端添加的学校数据
同时由于需要用到内嵌浏览器，所以需要先安装另一个Mod---[NplCefBrowser](http://wikicraft.cn/wiki/mod/packages/packages_install#/npl?id=8)
